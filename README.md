## Fenkod v0.1
Fenkod, fen bilimleri ile ilgili fonksiyonlar barındıran bir mini modüldür.

## Dökümantasyon
**caprazla fonksiyonu:**
sözdizimi:

    caprazla(string gen1, string gen2)
örnek:

    caprazla("Aa", "Aa")
çıktı:

    Aa Aa
    |   |
    AA Aa aA aa
##  
  **basinc fonksiyonu:**
sözdizimi:

    basinc(int f, int s)
örnek:

    basinc(10, 2)
çıktı:

    5.0

##  
  **pascal fonksiyonu:**
sözdizimi:

    pascal(int newton, int m2)
örnek:

    pascal(10, 2)
çıktı:

    5.0
##  
  **element fonksiyonu:**
sözdizimi:

    element(string an) # (atom number) max 30
örnek:

    element(1)
çıktı:

    Hidrojen (H)

##  
  **pH fonksiyonu:**
sözdizimi:

    pH(int value) # max 14
örnek:

    pH(7)
çıktı:

    Nötr
##  
  **CtoF fonksiyonu:**
sözdizimi:

    CtoF(int celsius)
örnek:

    CtoF(10)
çıktı:

    50.0 °F
##  
  **FtoC fonksiyonu:**
sözdizimi:

    FtoC(int fahrenheit)
örnek:

    FtoC(10)
çıktı:

    -12.222222222222221 °C
